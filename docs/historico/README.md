- Use estes arquivos apenas como memória técnica.
- Não trate estes arquivos como especificação vigente.

A documentação antiga misturava pipeline, sprint, tela executiva, regras específicas e decisões exploratórias. Isso fazia o MIDWAY parecer uma ferramenta focada em `92/82`, quando o objetivo atual é tratar todas as anomalias relevantes dos dados OMS/ADMS e gerar exportação governada para o IQS.
